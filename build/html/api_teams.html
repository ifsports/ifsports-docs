API de Times¶

Documentação para o serviço de gerenciamento de equipes e jogadores.

GET /api/v1/teams/¶

Get Teams By Campus

Query Parameters:

  • status ({'null', 'string'}) – Filtrar equipes por status

  • campus ({'null', 'string'}) – Filtrar equipes por campus

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error
