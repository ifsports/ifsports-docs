API de Calendário¶

Documentação para o serviço de agendamento e calendário de jogos.

GET /api/v1/calendar/auth/login¶

Get Auth Url

Endpoint para o frontend obter a URL de autorização do Google. O frontend deve redirecionar o usuário para esta URL.

Query Parameters:

  • user_email (string) – (Required)

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error

GET /api/v1/calendar/auth/callback¶

Auth Callback

Endpoint de callback. O Google redireciona o usuário para cá após o login. Este endpoint não deve ser chamado diretamente pelo frontend. O email do usuário precisa ser passado de alguma forma, o state é ideal para isso.

Query Parameters:

  • code (string) – (Required)

  • state ({'null', 'string'})

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error

POST /api/v1/calendar/events¶

Create Event

Cria um evento na agenda do usuário. Requer que o usuário já tenha autenticado.

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error
