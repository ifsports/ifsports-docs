API de Auditoria¶

GET /api/v1/campus/{campus_code}/logs/¶

Lista os logs de auditoria para um campus específico.

Retorna uma lista paginada de todos os logs de auditoria associados ao campus_code fornecido na URL.

Parameters:

  • campus_code (string)

Status Codes:

  • 200 OK –

GET /api/v1/campus/{campus_code}/logs/{id}/¶

Obtém um log de auditoria específico.

Retorna os detalhes de um único log de auditoria, identificado pelo seu ID.

Parameters:

  • campus_code (string)

  • id (string) – A UUID string identifying this log.

Status Codes:

  • 200 OK –
